# Testing a 90s Style Site

Welcome to the **90s Style Site** project! This repository is dedicated to exploring and testing the aesthetics and functionality of web design from the 1990s. 

## Live Demo

You can view the site live at the following link:

[Visit Site](https://refayatul.github.io/Hikma/)

## Features

- **Retro Design**: Experience the nostalgic look and feel of 90s web design.
- **Simple Navigation**: Easy-to-use navigation reminiscent of early web interfaces.
- **Responsive Layout**: While maintaining a retro style, the site is designed to be accessible on various devices.
